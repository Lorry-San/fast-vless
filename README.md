> 感谢Zedware Network以及Zedware本人提供的大力支持！

本项目是一个轻量级、一键式的安装脚本，支持：

- ✅ 安装 Xray 最新版本
- ✅ 支持 VLESS + Reality 模式
- ✅ 流媒体解锁检测
- ✅ 支持中转模式：你可以将本机设置为 VLESS 流量入口，自动中转到其他节点
- ✅ 自动生成 VLESS 连接链接，直接用于客户端导入
